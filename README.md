# Smart Aquarium Blynk

This project is an IoT based smart Aquarium using blynk to control and send notification.

## Features
- Reads tank temperature, TDS Value.
- Send notification if temperature/TDS reach threshold value.
- Automatic heater control based on threshold value.
- Control feed servo, LED, etc. using button in Blynk.

## Hardware Component
- ESP32
- DS18B20 temperature sensor
- LED
- Heater
- 2 Channel relays
- Motor Servo
